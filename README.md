# Replication Repository for Reppin' your Constituency? Urban-Rural Differences in Geographical Representation in Swedish Parliamentary Debates 

## Overview

This repository contains the data, code, and documentation required to replicate the analysis presented in my forthcoming paper in *Political Geography*.

## Repository Structure

```
├── data/
│   ├── raw/              # Original, unprocessed datasets
│   └── processed/        # Cleaned and analysis-ready data
│
├── code/
│   ├── 01_data_collection.R
│   ├── 02_geocoding.R      
│   ├── 04_descriptive_statistics.R
│   └── 05_statistical_inference.R
│
├── outputs/
│   ├── figures/          # Generated plots and visualizations
│   └── tables/           # Generated tables
│
├── README.md             # Project documentation (this file)
└── LICENSE               # License for reuse
```
