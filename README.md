# Replication Repository for Reppin' your Constituency? Urban-Rural Differences in Geographical Representation in Swedish Parliamentary Debates 

## Overview

This repository contains the data, code, and documentation required to replicate the analysis presented in my forthcoming paper in *Political Geography*.

## Repository Structure

```
├── data/
│   ├── raw/
│   └── processed/
│
├── code/
│   ├── 01_data_collection.R
│   ├── 02_geocoding.R      
│   ├── 03_descriptive_statistics.R
│   └── 04_statistical_inference.R
│
├── outputs/
│   ├── figures/
│   └── tables/
│
└── README.md
```
